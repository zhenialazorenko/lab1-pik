Цей проект реалізує простий текстовий редактор на платформі .NET з використанням технологій Windows Forms та WPF. 
Мета проекту — ознайомлення з принципами створення графічних інтерфейсів користувача, обробки подій та роботи з файловою системою.
Програма дозволяє створювати нові текстові файли, відкривати існуючі, редагувати текст і зберігати зміни у форматі .txt.
